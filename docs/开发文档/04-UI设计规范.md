# Mini-Todo - UI 设计规范

## 1. 设计理念

- **简洁现代**：采用扁平化设计，减少视觉噪音，**去除卡片边框**
- **高效交互**：快速添加、一键完成，减少操作步骤
- **视觉层次**：通过颜色圆点区分优先级和状态
- **透明美学**：固定模式下的毛玻璃透明效果
- **极简列表**：列表项无边框，使用圆点标识优先级

## 1.1 UI 组件库

- **组件库**：Element Plus
- **图标库**：Element Plus Icons（@element-plus/icons-vue）
- **禁止使用 emoji 图标**

## 2. 色彩体系

### 2.1 基础色板

```scss
// 主色调
$primary: #3B82F6;        // 蓝色 - 主要按钮、链接
$primary-light: #60A5FA;  // 浅蓝 - hover 状态
$primary-dark: #2563EB;   // 深蓝 - active 状态

// 背景色
$bg-primary: #FFFFFF;     // 主背景
$bg-secondary: #F8FAFC;   // 次级背景
$bg-tertiary: #F1F5F9;    // 第三级背景

// 文字色
$text-primary: #1E293B;   // 主要文字
$text-secondary: #64748B; // 次要文字
$text-tertiary: #94A3B8;  // 占位符文字

// 边框色
$border: #E2E8F0;
$border-light: #F1F5F9;
```



## 11. 图标规范（Element Plus Icons）

- 使用 Element Plus Icons 组件库
- 图标尺寸：16px / 20px / 24px
- 图标颜色跟随文字颜色或使用主题色
- 交互图标添加 hover 效果
- **禁止使用 emoji 图标**

### 常用图标映射

| 功能 | Element Plus 图标组件 | 尺寸 |
|------|------------------------|------|
| 添加待办 | `<Plus />` | 20px |
| 完成状态 | `<Check />` 或 `<Select />` | 16px |
| 通知提醒 | `<Bell />` | 16px |
| 固定模式 | `<Lock />` / `<Unlock />` | 18px |
| 设置 | `<Setting />` | 18px |
| 更多操作 | `<MoreFilled />` | 20px |
| 删除 | `<Delete />` | 16px |
| 编辑 | `<Edit />` | 16px |
| 置顶 | `<Top />` | 16px |
| 标签 | `<PriceTag />` | 16px |
| 归档 | `<FolderOpened />` | 18px |
| 关闭 | `<Close />` | 18px |
| 最小化 | `<Minus />` | 18px |
| 日历 | `<Calendar />` | 16px |
| 时间 | `<Clock />` | 16px |

### 使用示例

```vue
<script setup>
import { Plus, Bell, Lock, Setting, Delete } from '@element-plus/icons-vue'
</script>

<template>
  <el-icon :size="20"><Plus /></el-icon>
  <el-icon :size="16" color="#EF4444"><Bell /></el-icon>
</template>
```
