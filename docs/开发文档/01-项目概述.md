# Mini-Todo 桌面应用 - 项目概述

## 1. 项目简介

Mini-Todo 是一款基于 Tauri + Vue 3 开发的桌面待办事项管理应用，专为 Windows 平台设计，具有独特的窗口固定功能和现代化的用户界面。

## 2. 项目目标

- 提供简洁高效的待办事项管理体验
- 支持系统级通知提醒
- 实现窗口固定功能（忽略 Win+D，透明背景）
- 采用现代化 UI 设计
- 支持数据导入导出，方便迁移

## 3. 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | Vue 3 + TypeScript | 组合式 API，类型安全 |
| 状态管理 | Pinia | Vue 官方推荐状态管理 |
| 桌面框架 | Tauri 2.x | 轻量级跨平台桌面框架 |
| 后端语言 | Rust | 高性能，内存安全 |
| 数据库 | SQLite | 轻量级本地数据库 |
| 拖拽功能 | vuedraggable | Vue 拖拽排序库 |
| 通知插件 | tauri-plugin-notification | Tauri 系统通知插件 |

## 4. 核心功能

### 4.1 待办管理
- 创建、编辑、删除待办事项
- 支持一级子任务
- 优先级设置（高/中/低）
- 完成状态标记
- 拖拽排序

### 4.2 通知提醒
- Windows 系统通知
- 预设提醒时间（5/15/30 分钟）
- 自定义提前时间

### 4.3 窗口特殊功能
- **普通模式**：浅色主题，可拖拽移动
- **固定模式**：
  - 透明背景
  - 固定在用户指定位置
  - 忽略 Win+D（显示桌面）
  - 禁用关闭、最小化、拖拽

### 4.4 数据管理
- SQLite 本地存储
- JSON 格式导入/导出
- 数据迁移支持

## 5. 目标平台

- **主要支持**：Windows 10/11
- **测试环境**：Windows 11

## 6. 项目结构

```
mini-todo/
├── docs/                    # 文档目录
│   └── 开发文档/            # 开发相关文档
├── src/                     # Vue 前端源码
│   ├── assets/              # 静态资源

│   ├── utils/               # 工具函数
│   ├── App.vue              # 根组件
│   └── main.ts              # 入口文件
├── src-tauri/               # Tauri/Rust 后端源码
│   ├── src/
│   │   ├── commands/        # Tauri 命令
│   │   ├── db/              # 数据库操作
│   │   ├── lib.rs           # 库入口
│   │   └── main.rs          # 主入口
│   ├── Cargo.toml           # Rust 依赖配置
│   └── tauri.conf.json      # Tauri 配置
├── public/                  # 公共静态资源
├── package.json             # Node 依赖配置
└── vite.config.ts           # Vite 构建配置
```
